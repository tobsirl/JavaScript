## typeof
The `typeof` operator returns a string indicating what type the value is

## Converting data types
### Explicit type conversion
```js
String(42)
Number('42')
parseInt('42')
```

### Implicit type conversion (coercion)
```js
'1' * '2' = 2 // number
'10' + 20 = '1020' // string
```
